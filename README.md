# java-cli-maven-crypto-PBKDF2-hashing

## Description
A demo for PBKDF2 one-way hashing of
a password.

## Tech stack
- java
- maven

## Docker stack
- openjdk:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
